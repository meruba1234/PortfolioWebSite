git 導入以前の開発 (2025年10月〜2026年1月)

バージョン管理 (git) を導入したのは 2026年1月末。それ以前の約4ヶ月間は共有フォルダ上で直接開発しており、 その期間の成果はコミット統計に表れていません。ファイルのタイムスタンプと成果物から復元した記録:

  • 受注振分け表 Excel VBA (2025年10月〜11月) — 受注データを営業所・得意先別に振り分けるエクセルシステム。 複数 VBA モジュール (メイン処理 / 集計 / シート管理 / マスタ差分確認フォーム) で構成。現在も月次で改修を継続
  • 受注CSV取込 (2025年11月完成) — 振分け表が出力した CSV を販売管理 DB に登録する Python スクリプト初版。 既存システム (VB.NET) の金額計算ロジックを解読して移植。のちに 5,489 行の v3 に成長し、Prefect 基盤に統合
  • 入金消込処理 フェーズ1 (2025年10月〜) — 経理の入金消込を支援する Excel 集計の初版。 のちに SQL Server 連携 + 部分和探索の VBA システム (2026年1月完成) に発展
  • 仕様書・操作マニュアルの整備 — この時期から既に、成果物と同時に仕様書 / 操作マニュアルを書く習慣を確立

2026年1月末に上記をすべて git 管理へ移行。以降の開発履歴はコミットログで完全に追跡可能。
