経歴のあゆみ

未経験からプログラミングを学び始めて、現在は社内SEとして業務基盤を一人で構築・運用しています。 試行錯誤の連続でしたが、一歩ずつ積み上げてきた軌跡です。

2022年頃

ポリテクセンターでプログラミング基礎を学ぶ

職業訓練校 (ポリテクセンター) でプログラミングの基礎を学習。 HTML / CSS / JavaScript / Java / Android アプリ開発などをカリキュラムに沿って習得。

当時の成果物として、クイズアプリ QuizVoyage を Android で開発。 本ポートフォリオサイトの原型もこの時期に手書きで作成 (生成AIなし、純粋な手作り)。

入社〜2025年8月頃

社内SE として実務経験を積む

卸売業の社内システム部門に入社。販売管理システムの保守・改修や、現場からの問い合わせ対応を担当。

既存システム (VB.NET / SQL Server / VBA) のコードを読み解きながら、徐々に業務全体像を把握。

2025年9月〜2025年11月

既存システムの解読と最初の自動化

販売管理システム (VB.NET) のソースコードを分析し、複雑な金額計算ロジックを Python に移植。

11月、最初の本格的な自動化として「入金振分処理」を完成。 ベンダー依頼すれば数十万円かかる改修を、自社内で実現できる体制の第一歩となった。

2026年1月

異常受注検知システム / 入金消込 VBA

受注データの異常値を自動検知するシステムを、ゼロから1週間で初期版リリース。

同時期、経理向けの入金消込 Excel VBA システムも完成。 SQL Server 連携と 4,095通りの部分和探索で消込判定を自動化。 以降、4ヶ月にわたって継続的に改善。

2026年2月

評価シート最高評価 (5/5)

上長による業務評価で、業務遂行・自己啓発・主体性・勤怠すべて 5/5 の総合最高評価を取得。 「今後も貢献してほしい」「社内SEとして期待」というコメントを受ける。

2026年3月〜4月

自動化基盤の構築 (Prefect / Linux サーバー)

Windows PC ローカル運用から、Ubuntu Linux サーバー (Docker Compose + Prefect 3.x) への移行を計画・実行。

二重稼働期間を経て本番運用化。 nginx リバースプロキシ、watchdog 二段構え、ゾンビ run 自動 Cancel など、運用面の安定化施策まで一人で実装。

2026年4月〜現在

業務自動化ジョブの大量追加

整備した基盤の上に、各種業務自動化ジョブを追加。 EOS 受注処理 (4社)、ゼンショー EDI 発注、取引先A 追加返品処理、BI ツール連携、月次請求書作成など、 9ヶ月で 19本以上のジョブを構築・運用。

同時期に AI (Claude) を業務開発の基盤に組み込み、仕様書生成・コードレビュー・障害対応の速度を数倍に向上。 カスタムスキル (社内ドキュメントテンプレート、ログ実装指針など) を自作して継続的に拡張。

2026年6月 (現在)

業務改善実績の体系化

5W1H + 残業換算 + ROI で業務改善実績を継続的に記録する仕組みを整備。 過去の実績を匿名化してポートフォリオ化し、本サイトに公開。

直近9ヶ月の総コミット数は 432件、稼働ジョブ数 19本以上、 社内合計の月間削減時間 90〜140時間、年間人件費削減効果 200万円以上を達成。

これまでの学びと今後

未経験から始めて4年、社内SE として実務に取り組んで2年弱。試行錯誤の中で「コードを書ける」だけでなく、 「業務を理解して仕組みに落とし込む」「設計方針とドキュメントを並走更新する」「運用・障害対応も含めて責任を持つ」という エンジニアの幅を広げてきました。

今後も、AI を活用しながら業務基盤を磨き続けて、現場の「面倒くさい」を「自動」に変えていきたいと考えています。
